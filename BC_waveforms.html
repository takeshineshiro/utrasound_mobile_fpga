Sample behavioral waveforms for design file BC.v

The following waveforms show the behavior of altsyncram megafunction for the chosen set of parameters in design BC.v. For the purpose of this simulation, the contents of the memory at the start of the sample waveforms is assumed to be ( FFF0, XXXX, FFF0, XXXX, ...). The design BC.v has one read port. The read port has 1 words of 16 bits each. The output of the read port is registered by clock.

Fig. 1 : Wave showing read operation.

The above waveform shows the behavior of the design under normal read conditions. The read happens at the rising edge of the enabled clock cycle. The output from the RAM is undefined until after the first rising edge of the read clock. The clock enable on the read side input registers are disabled. The clock enable on the output registers are disabled.
